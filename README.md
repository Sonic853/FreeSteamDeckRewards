# FreeSteamDeckRewards

允许您“解锁” 512GB 的独家 Steam 社区个人资料捆绑包和独家虚拟键盘主题。

通过 [Decky Loader](https://github.com/SteamDeckHomebrew/decky-loader) 加载此插件来“解锁” 512GB 专属的独家 Steam 社区个人资料捆绑包和独家虚拟键盘主题。
